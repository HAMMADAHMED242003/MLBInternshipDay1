Student Performance Data Cleaning & Visualization

## Data Cleaning Steps:
The following data cleaning tasks were completed:

Checked the dataset for missing values.
Removed duplicate records (if any).
Renamed columns for better readability.
Created a new column Average_Score by calculating the average of subject scores.
Created a Performance column based on the average score:
    1. Excellent → Average Score ≥ 90
    2. Good → Average Score between 80 and 89
    3. Average → Average Score between 70 and 79
    4. Needs Improvement → Average Score below 70
Sorted and filtered the dataset where required.
Saved the cleaned dataset as cleaned_student_performance.csv.

Key Insights
1. Students who studied more generally got higher marks.
2. Students of all genders performed almost equally.
3. Most students scored in the middle range, while only a few had very low or very high scores
