***How file handling and JSON work together***

File handling and JSON are used to store data permanently.

In Python, JSON helps us save data in a file in a structured way (like lists and dictionaries). When we want to save data, we use json.dump(). When we want to read data back, we use json.load().

So basically, Python data is converted into a JSON file for storage, and then converted back into Python data when we read it.
