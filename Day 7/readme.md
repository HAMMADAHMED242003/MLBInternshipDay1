# Iris Flower Classification

## What is Classification?

Classification is a type of machine learning that predicts a category. In this project, the model predicts the type of an iris flower.

Examples:

* Setosa
* Versicolor
* Virginica


## Regression vs Classification

**Regression**

* Predicts a number.
* Example: Predicting a house price.

**Classification**

* Predicts a category.
* Example: Predicting the type of a flower.


## Evaluation Metrics Used

* Accuracy: Shows how many predictions are correct.
* Classification Report: Shows precision, recall, and F1-score for each class.
* Confusion Matrix: Shows the correct and incorrect predictions.


## Model Performance and Observations

* The model gave high accuracy.
* Most flowers were predicted correctly.
* The confusion matrix helped identify any wrong predictions.
* The classification report showed the model performed well for all classes.
